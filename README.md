# QRgen

Simple Vue based web application for creating payment QR codes in **PAY by square (Slovakia)** and **SPD (Czech Republic)** format.

## Highlights

- PAY by square and SPD payment QR codes
- option to select a tip for a new payment QR
- QR scanner in Chromium based browsers with IBAN and SWIFT validation
- SK / CZ / EN languages, light / dark mode, offline support

## Demo

Available at [qrgen.pa2lo.net](https://qrgen.pa2lo.net/)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
